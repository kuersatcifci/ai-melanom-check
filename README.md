# AI Melanom Check

Browser-basierte Demo zur Hautläsionsklassifikation auf Basis des ISIC/HAM10000-Datensatzes.

⚠️ KEIN Medizinprodukt nach MDR (EU) 2017/745. Kein Diagnose-Tool. Ausschließlich als Compliance-Demonstrator und Studienprojekt veröffentlicht.

## Tech Stack
- Next.js 16 (App Router, TypeScript)
- Tailwind CSS 4 + shadcn/ui
- ONNX Runtime Web 1.26 (Browser-Inferenz, keine Server-Übertragung)
